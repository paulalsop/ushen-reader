# CARDS_SPEC — 任务卡规范（v6 · 因果优先）

## 卡片模板（每章一张）
```
### 第NNN章 标题
- 案/段：所属单元案与功能段（A起/B承/C转/D合/E钩）
- 行程：上一站 → 本章所在 → 章末指向；注明原文「又东X里」；回程/快进须写明
- 戏剧问题：谁现在想得到什么？
- 核心选择：不可兼得的二选一（或明确「无选择，仅后果」并说明为何）
- 必达结果：1–2 句，本章必须发生的结果（不是待办清单）
- 状态变化：章末相对章初，人物/关系/资源/目标何者不同
- 持续后果：立即代价 + 将在何章兑现
- 线索增量：新观察改变了哪个旧判断；无则 —
- 出场：本章人物（含声音提示）；新人须写出场因由
- 知识用途：改变行动/造成误判/兑现后果；无则 —
- 喜剧发动机：人物行为如何自然产笑；无则 —
- 泪点：无则 —；有则一刀即止
- 雷：埋/爆 + 编号（新雷编号 Vx-Rnn）
- 结尾方式：决定 / 后果 / 关系变化 / 意象 / 强钩
```

## 全局铁律（每章自查）
1. 苏慕凡人锁死：不许法力/异能/升级；赢靠知识、嘴、勘误。
2. 时代红线：无书纸字钱官府书院考试客栈（见 `01_world/WORLD.md` §3）；校样只在苏慕脑海。
3. **每章必须有状态变化与选择/后果**；笑点、知识、强钩允许填「无」。
4. 章长 2800–4500 的预期；对话驱动。
5. 主角团不写死（苏慕/白耳/招摇/帝江/狗系统）；计划内死亡仅限 STORYLINES 标注者。
6. **书即行程单**：卷一锁定南山经䧿山系西→东；不得跳山、不得无故折返。已校之山再次路过=快进掠影，不重开整案。
7. 禁止解释笑点；禁止把同一谜底换说法连讲超过两章。

## 衔接规则
- 每卷第 100 章必须给出通向下一卷开场的事件钩。
- 卷内：每案须有一次由主角选择造成的不可逆损失；每案结尾还掉本案发出去的雷（或明确转账）。
- 新人物出场须与 CAST 一致；CAST 没有的角色标「新增-待导演批」。
- 并行写章以「行程字段 + LEDGER + 上章状态变化」为唯一承接；凭空加人、跳山、无必要折返 → 退稿。
