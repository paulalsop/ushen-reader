# WORLD — 大荒世界规则（v5）

## 1. 基本法
- 《山海经》所载为真：山川、异兽、草木、诸神、部落都在。**但文本与现实有出入**——每条出入都是一桩案子（"书不对"是全书的探矿权）。
- 地理即《山海经》十八篇：五藏山经 → 海外经 → 海内经 → 大荒经。行程按原文走（"又东三百里，曰堂庭之山"）。
- **没有修炼体系**。大荒的神异是天生地养的，不是练出来的：神是神，兽是兽，人是人。苏慕凡人锁死，全书不出现"灵气/境界/功法"。
- 长生存在但稀有（不死药、不死民），且必有代价——"不死"是全书最贵的题，卷四窫窳案第一次正告。

## 2. 山是活的，名是稀罕物
- 山有名，有名则有灵：山会睡、会翻身、会闹脾气（招摇=巨型熊孩子）。兽有灵者能言（白耳）。
- 名是稀罕物：山有，兽有，人不一定有。部落民以物相称（石头、尾巴、香）。
- 人间组织=部落+巫。巫管三样：草药、火塘、神谕（兽皮/木板上口耳相传的"老话"）。

## 3. 时代红线（防出戏，最高警戒）
**仓颉未造字，世界无书。** 人间只有刻画符号（陶符/木板/石壁古刻），无人能系统识读。
- 禁止出现：书、纸、字（除古刻）、钱、官府、朝廷、考试、书院、客栈、银两、官职、姓氏制度、军队、城池。
- 允许存在：部落、石器、骨器、陶、兽皮、结绳、篝火、穴居/石屋、渡口、原始交换（以物易物）。
- 苏慕的内心吐槽可用现代词（他是穿越者）；狗系统的营业腔合法（它本就不属于这个世界）——**现代梗只从这两个口子进**。

## 4. 「书不对」与深层异常（导演亲管，滴水渗透）
- 修订痕迹：地理/条目与文本不符；古刻与今传不符（神谕错草案）。
- 翻脸/翻页：局部世界被"改动"的现象（招摇失眠、某地凭空多出/少了东西）。
- 东边裂口：白猿世代向东静坐；人族部族东迁之谜；"天上有东西被撕开过"。
- 天狗（"天狗食日"的神话面）：与删改相关，卷四后启用，禁早。
- 规则：深层异常每卷至多给一次正面目击，其余只给痕迹和传说。

## 5. 神话=系统事件（涨知识主矿脉）
- 十日并出=冗余 bug；后羿射日=删九留一；女娲补天=打补丁；共工触山=宕机事故现场；大禹治水=重写地理；绝地天通=停笔+关闭上升通道。
- 使用规则：神话新解必须先梗后泪、服从当卷主题，不为炫技。
