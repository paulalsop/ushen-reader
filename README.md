## 正文硬锁

- 第001—100章是文风与叙事密度的首要锚点；第101—155章已按同一标准统一。
- 普通章正文原则上不少于3000字；重要冲突章以3500—4500字为宜。字数统计到第一个 `---`，不得靠章末自报充数。
- 润色不能以压缩为目标。应补足具体场景、人物动作、关系变化、有效伏笔和可兑现后果。
- 世界内无纸、无通用文字、无钱、无官府、无书院。结绳、刻痕、泥形和实物提示都不能自行讲出完整句子，须由本地持有人或现场见证者口述。
- 系统默认私语；开启共听必须明确写出声道切换。共听只传系统主动发言，不泄露苏慕心念、记忆与残册内容。
- 每章文件名、首行章号与标题须一致；保留【章末自报】供生产使用，发布正文时剔除。
- 并行写作时一章一提交，只暂存本章文件；不得覆盖或合并其他agent尚未完成的修改。
