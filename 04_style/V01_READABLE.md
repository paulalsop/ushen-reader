# 卷一可读宪法（V01_READABLE · 读者优先）

> 本轮最高优先。覆盖并收紧 `V01_POLISH.md`。目标：**普通读者一口气读得懂、愿意笑、想翻下一章**。  
> 不是机械删模板，是写成一流可读小说。

---

## 〇、改章前强制流程（Agent 必做）

对每一章，先当读者读一遍，在心里（可写在回复里）打分：

| 项 | 1–5 分 | 找茬问题 |
|----|--------|----------|
| 一眼懂 | | 有没有要猜的句子？ |
| 人话对话 | | 像不像真人会说的话？ |
| 想笑/想跟 | | 有没有具体场面让人嘴角动一下？ |
| 知识落地 | | 「校样/条目」有没有白话垫一句？ |

**任一项 ≤3：必须改。** 改完再自测：删掉上下文，单句扔给路人，路人能不能懂。

## 一、硬禁（出现即退稿）

### 1. 谜语体叙述
禁止读者要停下来猜的「聪明短句」。

坏例：
- 「渡口不宽。宽的是蛇。」
- 「动不是扑，是抬步。」
- 「错会吃人。」
- 「该吃之前，先把账立住。」
- 「他会记。我会搬。」／「会拽。」（无上下文说明拽什么）

好例：
- 「渡口本身就两步宽，更糟的是石梁上趴满了蛇，人几乎没处下脚。」
- 「那头鯥动了——不是扑过来咬人，只是慢吞吞抬腿往东走。」
- 「字写错了，人就可能死在错路上。」
- 「先把规矩说清楚再吃，免得白猿以为我们欠它们人情。」
- 「名字我来记。东西我来搬。」／「怪木会回握，一碰就拽住人的手。」

### 2. 「A不是B，是C」强行纠偏句
整章这类句式 **≤1 次**。直接写它在干什么，别先否定读者脑子里还没出现的画面。

### 3. 电报人 / 审计腔对话
人物不许说成指示牌。

坏例：苏慕「叫就听。听完再决定跑不跑、打不打。」  
好例：苏慕阴恻恻：「它要是还叫，你就滚出去压它们——你不是一座山吗？我觉得这办法妙。」（性格：怂里带损、职业病吐槽，但**句子完整、有口气**）

### 4. 行话裸奔
「校样」「条目」「附注」「KPI」「核销」等：
- **第一次出现**必须旁边白话：如「脑子里那本没印完的《山海经》校稿」。
- 或干脆改成读者一听就懂的词：「书上那行字」「原文写的」。
- 禁止连续三句只扔行话不解释。

### 5. 事实漂移仍禁
地名、物件、伤势、关系裂痕、校勘进度、雷线结果不得推翻。  
**允许**：在既定结果内增加**好笑的具体原因与场面**（例如「不可以上」实地发现是滑到离谱、一爬就摔——只要最终仍是绕行/横切，不登顶）。

### 6. 越界
只改本批章节；不动 `101+`、LEDGER、CAST；不 git commit。

## 二、对话：正常、有个性、能出梗

| 角色 | 说话像 | 禁止 |
|------|--------|------|
| 苏慕 | 社畜编辑：会贫、会损、会认账；危急时句子仍完整 | 三个字一刀切；端着讲大道理 |
| 白耳 | 话少但**说人话**；冷归冷，指代清楚（拽什么、记什么） | 单字谜语；让读者猜动词宾语 |
| 招摇 | 熊孩子显摆；可以「本座」但别刷屏；怕错过时会露怯 | 每句补丁「——」；正经哭场贫嘴不停 |
| 系统 | 客服腔漏风、心虚装死；被戳破时破音 | 同梗复读；长篇 KPI 播报 |

**验收**：盖住名字，五句对话能听出是谁；且路人不用猜宾语。

## 三、笑点：要大笑、要意外、要长在事上

优先顺序：
1. **意想不到的实地原因**（书上「不可以上」→ 爬上去脚底像抹了油，摔得生不如死）
2. **身份错位**（用改稿逻辑对付神话；神兽当业务办）
3. **误会链滚雪球**（比口头禅有用）
4. **白耳一句冷刀子** / **系统心虚破功**

铁律：
- 笑话删掉旁白仍成立。
- 一章至少一个「愿意截图发群」的具体场面（动作或对话），严肃章（裁切/托付）可改为「一刀余味」代替大笑。
- 禁止解释「这件事很好笑」。

**重点章加戏（本轮点名）**
- `010 不可以上`：加大苏慕↔狗系统斗法趣味；给「不可以上」一个意外又合理的实地理由（滑/摔/山会翻脸踢人等），与绕行横切不矛盾。
- 开篇公路段（006–020）：破谜语体，加人话与笑点。
- 041 鯥同行：删「动不是扑」类句，改成画面直接可见。

## 四、叙述：通畅第一

- 短句可以，但**信息要全**：谁、对什么、做了什么。
- 比喻只能帮理解，不能代替事实。
- 专业动作（翻校样）写成：「他闭眼，脑子里翻开那本《山海经》校稿——穿越前改了七遍、还没印成书的那本。」
- 转场后第一句承接场面，不开会、不谜语开场。

## 五、交付

1. 直接写回 `章节/第XXX章_*.txt`
2. 保留章末【章末自报】；事实栏不发明；措辞可随正文变清楚
3. 回复格式：
   - 每章读者分（四项）
   - 找茬（别扭句原文→改法）
   - 本批最大笑点/加戏一览
   - 自检：谜语体清零？行话有白话？对话像人？

字数：勿灌水暴涨，勿砍剧情；可读优先于凑字。
