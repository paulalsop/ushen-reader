# STYLE — 写作风格定义（v7 · 因果优先与信息边界）

## 一句话基调
**世界是真的，人是不靠谱的。**
神话认真辽阔，角色全体沙雕；笑要无厘头，泪要后知后觉。
定位：山海经悬疑公路轻喜剧——每章可以好笑，但必须有选择、代价与状态变化。

## 一、笑点机制（工具箱，不是配额）

五类可用，**不强制每章凑数**：

1. **反差**：神话人物先找不正经的那一面。
2. **拆解崇高**：大事件有小细节，大人物有小毛病。
3. **职业病**：苏慕编辑脑、狗系统营业腔、古人逻辑撞现代常识。
4. **复读梗**：同一动作/口癖再次出现时，必须升级含义或转成泪点；原样复读不算。
5. **误会链**：靠信息差滚雪球，不靠解说。

铁律：
- 笑话必须长在人物目标与行动上；删掉旁白解释后仍成立。
- **禁止解释笑点**：不许写「这件事很好笑」「记忆点捕获」「结构评分」之类标注。
- 一章可以无笑点；严肃章不得为了过门硬塞吐槽。
- 同一口头禅跨章高频复用即退稿（尤其狗系统「感谢您的理解／重复领取概不补发」）。

## 二、泪点机制（双层梗）

- 先笑，后沉默；一次只给一刀，给完立刻翻篇，不许煽情段落。
- 泪点三味：来不及、回不去、甩不掉。
- 禁止：哭腔旁白、「他流下了眼泪」式直给、连续两章放泪。

## 三、知识机制（必须改变行动）

- 真《山海经》条目随剧情变现：先遇事，后出书。
- 苏慕可以背原文，但必须用在刀刃上——背到一半被打断，比背完更好笑。
- **知识必须会错**：信错条目，付真代价；校对自己也会制造新后果。
- 单纯背书、换说法复述，不算知识推进。无知识章允许（填「无」）。

## 四、叙事硬门（比笑点更重要）

每章必须同时满足：

1. **即时目标**：谁现在想得到什么？
2. **阻力意志**：阻力来自另一个人的意志或环境反制，不是作者暂不公布信息。
3. **选择**：主角或关键角色作出不可兼得的选择。
4. **状态变化**：章末与章初相比，人物/关系/资源/目标至少一项不同。
5. **持续后果**：本章选择立即付出代价，或明确将在后续章兑现。

「看见异常并继续东走」不算状态变化。
「又发现一条线索」若未改变判断或行动，不算推进。

## 五、语言规矩

- 短句为主，长短交错；对话驱动；叙述人不端着。
- 现代梗只许从两个口子进：苏慕的脑子、狗系统的嘴。古人嘴里不出现代词。
- 系统声道必须可判：写明“脑内／心里／私语”即只有苏慕能听；写明“共听／外放／从眉心传出”即百丈内同行者可听。古人不得回应私语；共听只传系统主动发言，不泄露苏慕心念、记忆与残册页面。群体场景首次切换声道必须落一笔提示。
- 白耳不读心、不全知。它只能报告亲历、确知来源与尚存记忆；记忆到哪里结束，必须说清到哪里。招摇也不能因山神身份读取苏慕心念。
- 人物复述重要消息时，优先区分：亲眼所见、明确来源的听说、猜测、不知道。发现说错须当场承认，不得用新故事悄悄覆盖旧话。
- 结绳、刻痕、骨符、泥形、脚印与实物提示都不是通用文字。它们只能提醒停步、指认对象或帮助当事人记忆；完整含义必须由本地持有人、制作人或现场见证者口述并允许对方回述。
- 世界内交换写兽牙、栗粮、盐、藤席、劳力等具体物，不写售价、收费、高价、购买、退货等现代商业表达。可以有摊、有交换、有欺骗，但必须长在时代内的物与人上。
- 禁止：网络烂梗直给、屎尿屁、段子合集化。
- 普通章正文原则上不少于3000字；重要冲突章3500–4500字。统计到第一个 `---`，不含章末自报。润色不得以缩水为目标。
- 扩写只补有效内容：具体动作、空间关系、人物选择、关系变化、后续会兑现的伏笔。禁止同义反复、审计报告体和机械凑字。
- 少用机械的“A不是B，是C”；需要纠偏时尽量让事实、动作或他人反应自己完成。
- 结尾方式任选其一：决定 / 后果 / 关系变化 / 余味意象 / 强钩。**不强制章末悬念句**；连续两章不得用同一类空悬念收尾。

## 六、格式与标题

- 文件名统一为 `第NNN章_标题.txt`；首行统一为 `第NNN章 标题`。除文件名因兼容性省略的标点外，章号与标题用字必须对应。
- 标题应来自本章最具体的冲突、动作或关系变化，避免“方案一”“经济学”“风波”“报告”等制作腔。
- 正文只保留一个完整版本；禁止重复段、残段、多个结尾并存。
- 第一个 `---` 后为【章末自报】，不计入正文篇幅；发布时剔除，生产稿必须保留并与正文事实一致。

## 七、幽默红线

- 不嘲笑苦难：可以笑神的架子，不笑人的穷苦。
- 不拿泪点当笑点：双层梗的下层永远是真的。
- 苏慕可以怂、可以贫，不许贱；狗系统可以狗，不许坏。

## 文风锚点
第001—100章是首要呼吸、密度与人物声音锚点；第101—155章已依同一标准统一，可作为卷二连续性锚点。后续正文不得突然转为提纲体、审计体或短章压缩体。
