# 《有神》写作技巧手册

> 用途：把 CORE、WORLD、CAST、STYLE、SOP 中分散的规则，整理成可直接用于构思、写章和改稿的操作指南。

## 一、先抓住这部小说真正的发动机

《有神》不是普通的“穿越者靠知识碾压异世界”，而是：

**一个熟悉《山海经》文本的编辑，进入文本所描述的真实世界；他每次用书本解释现实，都可能救人，也可能犯错。**

因此，每个有效故事都应包含下面这条链：

1. 苏慕遇到一个《山海经》条目可以解释、却解释不全的现实。
2. 他先依赖书本形成判断。
3. 现实中的人、兽、山以自己的意志反驳这个判断。
4. 苏慕必须在两种都有代价的做法中选择。
5. 选择改变人物关系、资源、道路或对世界的理解。
6. 新结论不是“原文全错”，而是“文字只记下了现实的一部分”。

最重要的题眼不是勘误本身，而是：

**条目写的是类别，苏慕面对的是具体的人。编辑文字容易，编辑别人危险。**

## 二、单章的基本结构

### 1. 开头：先给眼前麻烦，不先讲设定

开篇尽快回答三个问题：

- 谁现在想做什么？
- 什么东西立刻拦住了他？
- 如果失败，他会马上失去什么？

好的开头是具体动作，例如迷路、求食、过河、护送、传话、救人。不要用长段世界观、神话原文或抽象异象开会。

### 2. 中段：让原计划失效

转折最好来自人物意志或环境反制：

- 神兽拒绝配合，因为它有自己的尊严、秘密或误解。
- 山不是背景，而会睡、翻身、闹脾气。
- 条目写对了结果，却漏掉实现结果的过程。
- 苏慕掌握的版本已经被修订，正确知识反而把他带进危险。
- 狗系统提供的帮助不完整、过时或附带代价。

不要把“作者暂时不告诉答案”当阻力。真正的阻力必须能够主动做事。

### 3. 高潮：逼出不可兼得的选择

选择不能只是“查还是不查”“走还是不走”。两边都要损失某种重要东西，例如：

- 公开校正传说，可以避免更多人受骗，却会伤害靠传说生存的神兽。
- 救眼前的人，就会失去追查归途的唯一线索。
- 强行解释符号，可以更接近答案，却可能污染尚未成熟的文字。
- 带招摇离山能保住村子，却会让招摇承担离山代价。

选择完成后，故事必须关上一扇门。资源用掉、信任受损、秘密公开、道路断绝、关系改变，都比单纯“又知道了一点”更有分量。

### 4. 结尾：落在变化上

结尾可落在五类变化之一：

- 决定：人物决定下一步做什么。
- 后果：本章选择立即反咬。
- 关系：两个人从提防变为欠情，或从同路变为分歧。
- 意象：用一个具体画面留下余味。
- 强钩：出现一个明确动作、物证或来客。

避免用“更大的秘密正在逼近”“他不知道命运已经改变”这类空悬念。

## 三、苏慕的独特解题法

苏慕的优势不是法力，而是编辑能力。写他解决问题时，优先使用以下手段：

### 1. 校对定义

分清原话究竟描述了什么，没描述什么。例如“佩之不聋”可能记录的是信息能够抵达，而不是听觉被治愈。

### 2. 比对版本

把口耳传说、古刻、现实结果和脑内残册放在一起，寻找增加、删减、误读与转述变形。

### 3. 追查因果链

不满足于“有效”，继续问：是谁做的、如何做到、代价由谁承担？《有神》的知识趣味往往藏在“结果正确，过程被漏记”里。

### 4. 克制解释

编辑的成熟不只是能解释，更是知道什么时候不该解释。面对人物的名字、创伤或尚未成形的符号，苏慕应避免把自己的版本强加给别人。

### 5. 用语言和流程制造行动

他可以谈判、立约、做对照、复述全句、公开勘误、设计验证，但不能突然会战斗、制造现代科技或成为全知军师。

检验方法：把苏慕换成任意聪明人，如果解法仍完全成立，就还没有写出“编辑主角”的不可替代性。

## 四、《山海经》知识怎么写才不成讲课

知识只在三种情况下进入正文：

1. 改变行动：因为原文信息，人物选择某条路或某种办法。
2. 造成误判：苏慕相信条目，因此付出真实代价。
3. 兑现后果：前文提过的性质，在危机中突然变得有用或危险。

推荐顺序是：

**先出现怪事 → 苏慕想起原文 → 用白话解释关键点 → 采取行动 → 现实修正原文。**

原文引用宜短，并允许被动作或人物打断。第一次出现“校样、条目、附注”等编辑术语时，要紧跟白话解释。古人和神兽不能像读过《山海经》一样接话。

## 五、轻喜剧怎么长在剧情上

这部小说最适合的喜剧不是段子，而是“神话身份与具体办事方式错位”。

### 1. 崇高外壳，小毛病内核

- 山神是怕错过热闹的熊孩子。
- 祥瑞像催婚办工作人员。
- 神兽的神效可能靠辛苦传话完成。
- 凶兽散布凶名，只因为社恐不想迎客。

人物的小毛病必须与他的目标有关，不能只是贴一个搞笑标签。

### 2. 职业病碰撞神话

苏慕在危险时仍想挑定义、查版本、纠正歧义；狗系统用漏风客服腔处理生死事故。笑点来自他们认真办错位的事。

### 3. 误会链滚雪球

让一句话在不同角色的目的中逐步变形。每次变形都要产生新行动，不能只换一种说法重复同一个笑话。

### 4. 复读必须升级

同一句口头禅再次出现时，应改变意义：第一次好笑，第二次成为关系暗号，第三次可能变成泪点。没有升级的复读应删除。

### 5. 笑后只落一刀

先让读者笑角色的古怪，再用一个事实揭出古怪背后的“来不及、回不去、甩不掉”。泪点说一次即可，不解释，不安排长篇哭诉。

## 六、人物声音

### 苏慕

完整句子，社畜编辑式的贫与损；嘴上吐槽，行动认账。危急时会抠字眼，但不能端着讲大道理。他最重要的内在恐惧是：把活人也当成可以修改得更顺的稿子。

### 白耳

话少、冷、准确，省字但不能省宾语。它知道很多，却不靠谜语腔吊读者。名字对它很贵重，所以不随便直呼全名。

### 招摇

熊孩子式显摆、好奇、拆家式帮忙。“本座”可以点味，不能刷屏。真正怕错过或怕沉睡时，应让稚气露出来，而不是继续硬塞笑话。

### 狗系统

客服腔漏风、心虚装死、偶尔意外可靠。它不能全知，不能替主角解谜，也不能连续复用同一营业口头禅。它最动人的底色是一直等一个不会再来的指令，但前期只许露痕迹。

### 神兽与神话人物

设计时使用三层法：

1. 《山海经》记载的外壳。
2. 一个与外壳形成反差的具体怪癖。
3. 怪癖下面不可取笑的真实处境。

只有前两层会变成段子人物；只有第一、三层又会过于端重。

## 七、悬疑与长线控制

### 1. 大谜同时不超过三个

读者当前需要持续记住的大问题宜控制在三个以内。小案应有始有终，不能只负责不断加问号。

### 2. 线索必须改变判断

有效线索链是：

**观察 → 修正假设 → 据此行动 → 行动带来新风险。**

只重复诡异气氛、换说法重讲同一谜底，不算推进。

### 3. 每案结清一部分账

单元案可以保留更大的世界谜团，但本案的人物问题、误会或规则问题必须得到明确答案。答案揭开后，后续应转入后果，不要五章内反复揭同一个答案。

### 4. 深层异常要稀

翻页、裁切、作者痕迹、系统来历等高层异常，应以物证和后果渗透。频繁正面展示会迅速消耗神秘感。

## 八、语言与可读性

1. 短句为主，但一句要有完整信息：谁对什么做了什么。
2. 对话推动行动，内心戏只保留判断、犹豫和选择。
3. 比喻帮助读者看见画面，不能代替事实。
4. 少用“A不是B，是C”的聪明纠偏句；整章最好不超过一次。
5. 禁止电报腔、谜语体、审计腔和无宾语动词。
6. 现代词只从苏慕内心与狗系统嘴里进入。
7. 章末自报属于制作资料，发布正文时必须剔除。

特别警戒：禁止把某个字当连接词机械接龙，例如连续使用“处”“处处”“写处”“改处”串接全文。这会破坏句法、画面和人物声音。重复、残段、同一结尾多次改写，也属于未完成草稿，必须整章重写，不能局部润色掩盖。

## 九、常见失败模式

- 主角发现异常，但没有作出选择，最后只是继续赶路。
- 知识只用于展示作者查过资料，没有改变任何行动。
- 每个角色都说短促谜语，盖住名字后听不出是谁。
- 同一创伤被连续解释，泪点变成角色说明书。
- 为了轻喜剧，在严肃场面硬塞系统吐槽。
- 阻力只是信息没公布，没有任何角色主动阻拦。
- 校勘永远正确，主角变成知识碾压型金手指。
- 正确校勘没有社会后果，世界像等待主角修改的死稿。
- 章节堆了许多新名词，却没有一个具体场景可记。
- 强钩只说“还有秘密”，没有动作、物证或具体危险。

## 十、写章操作流程

### 写前

1. 读连续性账本、最近两章和相关人物卡。
2. 用一句话写清本章戏剧问题。
3. 写出二选一的核心选择，以及每边会失去什么。
4. 确认本章只推进一个主要事件。
5. 决定知识的用途：改变行动、造成误判、兑现后果，或不用知识。

### 初稿

1. 用具体麻烦开场。
2. 让人物目标互相冲撞。
3. 在中段使原计划失效。
4. 让苏慕用编辑方法重新判断。
5. 逼出选择并立刻落下一个后果。
6. 用决定、后果、关系、意象或具体强钩收尾。

### 改稿

逐项检查：

- 普通读者能否一遍读懂每句话？
- 对话盖住人名后，能否听出角色？
- 笑点删去旁白解释后是否仍成立？
- 《山海经》知识是否真正改变剧情？
- 本章是否有不可兼得的选择？
- 章末状态是否与章初不同？
- 这个变化是否会影响下一章？
- 是否重复了读者已经知道的谜底？
- 是否出现时代越界、事实漂移或角色禁句？
- 是否有机械接龙、残句、重复段落或多个版本未清理？

## 十一、最简写章公式

如果只记一套公式，就记下面这套：

> **具体麻烦出现 → 原文给出一个看似可靠的答案 → 人物意志使答案失效 → 苏慕在两种代价之间选择 → 校正文字，也被现实校正 → 留下一个不可逆后果。**

这套结构能够同时承载《有神》的知识、悬疑、喜剧和题眼。
